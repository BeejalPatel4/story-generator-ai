
# story-generator-ai


Story Generator AI is a full-stack web application built with a React frontend and a FastAPI (Python) backend that generates creative stories using AI. The backend follows a clean, scalable architecture with separate layers for routers, schemas, models, and database management, while the frontend provides an intuitive user interface for submitting prompts and viewing generated stories. This project demonstrates modern full-stack development practices, API-driven design, and AI integration.
